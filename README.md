# OpenGL Renderer

This is a simple toy OpenGL renderer made in C, using my in developement engine.

[<img width="486" height="289" alt="image" src="https://github.com/user-attachments/assets/ee1e7a88-e40b-4821-8bf8-00e4623d9dda" />](https://www.youtube.com/watch?v=Y87ZzacBAQ8)

## Features
- PBR inspired material rendering
- Directional light
- Directional shadows
- Volumetric fog
- Debug normal, albedo, depth channels

## Engine Features
- Hot-reloading of the "game" DLL
- Arena allocator

## Dependencies
- OpenGL
- GLFW
- stb_images
- cimgui (a Dear ImGui binding for c)
- cglm

## Ressources

The materials come from Poliigon and the skybox from Polyhaven
